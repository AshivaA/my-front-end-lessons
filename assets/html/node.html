Node-page


explanation:
-Node.js alow us to run javascript code outside of browser.
-actually we use node to make command-line (terminal) applications.
(node/package.json/node_modules )installation:
1-install node in your computer from ( Node's website ).
guid to installation: https://coding-boot-camp.github.io/full-stack/nodejs/how-to-install-nodejs

2- check if it is install:
a-open terminal
b- write (node -v)
c- this is the version of your node (v18.17.1)

3- create( {} package.json ):
a-open terminal(in your related folder)
b-write ( npm init -y )

-package.json holds important information about your project(ex:project name) and a list of dependencies required by the application.

4-create( node_modules folder/ gitignore ):
a- goes to ( https://www.npmjs.com/ )
b- search any package you want to(ex:inquirer)
c- goes to install, copy it and past it in terminal (ex:npm install --save inquirer)
-now you have ( node_modules/package-lock.json )
d- create gitignore file:
1- into vs code goes into the root (where we have readme and index.html/index.js)
2- open terminal and write (touch .gitignore)
3- open the gitignore file and write (node_modules)inside the file.
4-in terminal write this comments:
-git status( to check that node_modules is not exist anymore in your github. )
-git add .
-git commit -m "added gitignore"
-git push origin main


how to uninstall the package?
npm uninstall package name( ex: npm uninstall inquirer )

important:
-we should never use node.modules in github so we need to make .gitignore file in project root where we have readme and index.html/index.js

5- to execute the code you write in your( index.js) open terminal and write:
node index.js
ex:
1- in index.js file ->( console.log("hi"); )
2- to see ("hi") in terminal write -> ( node index.js )
import/export modules in node:
a- require(): (use to import(enter) data/modules into another page)
b- module.exports: (use to export data/modules from a page)

ex: we want to execute indexOne.js codes in another js file( here is indexTow.js file.)
so we need 2 js files:

1- indexOne.js (first file)
const name= ["Shiva","Amin"]
module.exports= name (export from this page)

2- indexTow.js (second file)
const result= require("./indexOne") (import to this page)
console.log(result);

3- in terminal write node indexTow.js
---
How to export tow data/modules :
const name= ["Shiva","Amin"]
const age= [38, 39]

module.exports= {name,age}
file access using file system(fs)-modules:
-using fs modules we can read/write/appending files .
for using all kind of fs modules we need to import fs modules first:
const fs= require ("fs");
--
Reading files:
1-import fs modules-> const fs= require ("fs");
2-syntax:
fs.readFile("./fs.csv", (error, data) =>
error? console.error(error): console.log(data.toString())
)

explanation:
-("./file.csv" )->(this is the file address. the csv is like txt to make a text)
-(error? console.error(error))-> ( ?means-> if that was error return error)
-(: console.log(data.toString()))-> ( :means-> if not shows data)
---
Writing files
-fs.writeFile() method use for asynchronously write data.
-fs.writeFileSync() method use for synchronous write data.

we can: replace data in existing file / add new data to existing file/ or make a new file whit new data
a- replace data (in existing file) by fs.writeFile()/ fs.writeFileSync()
b- add data to existing file by fs.appendFile()
c- make a new file with new data by just write new file name.

a-replace data:
1-import fs modules-> const fs= require ("fs");
2-syntax
fs.writeFile ("./existingFile.csv", " the text we want to replace in existing file " , (error) =>
error? console.error(error): console.log("success")
)
--

b-add data to existing file:
1-import fs modules-> const fs= require ("fs");
2- syntax
fs.appendFile("./existingFile.csv", " \n add new text to existing file " , (error) =>
error? console.error(error): console.log("success")
)

explanation: \n (make a text in new line)
--

c-make a new file with new data:
1-import fs modules-> const fs= require ("fs");
2- syntax
fs.writeFile ( "./newFileName.csv" , " new text in new file ", (error) =>
error? console.error(error): console.log("success")
)
